### Installation
1.Clone the repository

```bash
https://github.com/johnbabu021/quizzy.git
```
2.Change   your directory
```bash
cd quizzy
```
3.Install all packages
```bash
npm install
```
4.run the project

```bash     
npm start
```


All set
